## 第四章单片机的C51语言

## 一. 简答题（共3题）

### 1. (简答题)指出下面程序的语法错误： 
```c
#include
main(){ 
	a=C; 
	int a=7,C 
	delay(10) 
	void delay(); 
	cgar i; 
	for(i=0; i<=255; i++); 
}
```

没有#include<reg51.h> ，没有函数的返回值类型void，未声明变量就使用，cgar拼写错误，应该为char，函数声明在Main（）中，应该在外部声明，没有初始化变量c

### 2. (简答题)将片内RAM中20H、21H单元存放的无符号整数比较大小，并将大数存放到片外RAM的30H
```c
#include<reg51.h>
void main(){
    unsigned char a,b,max;

    a=*((unsigned char idata *)0x20)
    b=*((unsigned char idata *)0x21)

    if(a>b)
        max = a;
    else
        max=b;
	
    *((unsigned char xdata *)0x30)= max;
	
    while(0);
}
```

### 3. (简答题)将片外RAM中30H、31H单元存放的带符号整数比较大小，并将大数存放到片内RAM的20H
```c
#include<reg51.h>
void main(){
    signed char x= *((signed char xdata *)0x30);
    signed char y =*((signed char ydata *)0x31);

    *((signed char idata *)0x20) = (x>y)?x:y;
    while(0);
}
```

## 第7章单片机的串行通信技术
## 一. 简答题（共1题）

### 1. (简答题)两个80C51（主机、从机）晶振频率为12MHz，通信波特率为2400bps，请绘制流程图并编程实现。 
（1）主机将指令帧0xAF发向从机（高四位为主机地址，低四位为命令），从机对指令帧进行奇偶校验，正确则执行指令帧，否则通知主机重发指令帧； 
（2）从机判断指令是否为0x0F，若是，则将从机片内RAM中20H-30H中的无符号整数进行叠加（结果不溢出），结果存入从机的片外RAM中的35H，同时将结果的低四位组成数据帧0xBX（高四位为从机地址，X为结果低四位），发回主机； 
（3）主机接到从机数据帧后，将0x0X存入主机片内RAM55H。 要求主机通信以查询方式，从机通信以中断方式。
