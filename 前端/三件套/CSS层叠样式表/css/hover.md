在 CSS 中，要实现当鼠标悬浮在一个元素上时触发另一个元素的样式变化，可以通过以下几种方式实现

| 结构                 | 说明                 |
| ------------------ | ------------------ |
| `A:hover B`        | A 悬浮时，改变子元素 B 的样式  |
| `A:hover + B`      | A 悬浮时，改变紧邻的兄弟元素 B  |
| `A:hover ~ B`      | A 悬浮时，改变其后所有兄弟元素 B |
| JavaScript + Class | 任意元素，通过 JS 动态添加类   |
